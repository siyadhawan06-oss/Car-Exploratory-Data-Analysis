# Car-Exploratory-Data-Analysis

## About This Project

This project was completed as part of my Data Analytics Internship. The objective was to perform Exploratory Data Analysis (EDA) on a car dataset and identify meaningful patterns, trends, and insights.

## What I Did

- Loaded the car dataset using Python and Pandas
- Explored the structure of the dataset
- Calculated basic statistics such as mean, median, and count
- Analyzed car prices and mileage
- Identified the most common car brands
- Found the highest and lowest priced cars
- Summarized key observations from the dataset

## Tools Used

- Python
- Pandas
- VS Code
- Excel

## Key Findings

- Calculated average and median car prices
- Analyzed mileage distribution
- Identified the most common car brands
- Compared expensive and affordable cars in the dataset

## Conclusion

This project helped me understand how to explore and analyze data using Python. Through EDA, I was able to identify trends, summarize statistics, and gain valuable insights from the dataset.
